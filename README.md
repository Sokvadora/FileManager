# FileManager
Sencha Ext.js + Java Spring boot + H2 database
***
Before start:   

in 'webapp' directory: 
- `  npm install `
- ` npx sencha app build -dev `
- run spring boot app

***
#### Database H2  http://localhost:8081/h2-console
The application run on port ` 8081`

` you can change the port in the  application.yml`
